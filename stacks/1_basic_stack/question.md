# Question 1 Implement a Stack (Full Program)

Write a complete program that:

**1)** Defines a stack using a struct with an array, a top index, and a maximum capacity.

**2)** Implements the functions push, pop, peek, and printStack.

**3)** In main, create a stack of size 5, push the numbers 10, 20, 30, peek, print the stack, pop one element, peek, and print the stack again.
